ConfigORM - A simple configuration library.

Functions

load_config()

Load configuration data from all the sources.

This method loads the configuration data from all the sources specified during the initialization of this class. The configuration data is merged together and returned as a single ConfigSchema object.

Returns:
  • ConfigSchema( ConfigSchema ) –

    The loaded configuration data.
reload_config()

Reload configuration data from all the sources.

This method reloads the configuration data from all the sources specified during the initialization of this class. The configuration data is merged together and returned as a single ConfigSchema object.

save_config()

Save configuration data to all the sources.

This method saves the configuration data to all the sources specified during the initialization of this class.

Raises:
  • PermissionError –

    If one of the sources is read-only.
