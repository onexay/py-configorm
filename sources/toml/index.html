This module contains the TOMLSource class, a class for a TOML configuration source.

TOML (Tom's Obvious, Minimal Language) is a minimal configuration file format that is easy to read due to obvious semantics.

This module is part of the configorm package for handling configuration data.

Classes

TOMLSource

Bases: SourceBase

Class for a TOML configuration source.

This class is a subclass of SourceBase and represents a TOML configuration source. It provides methods to load and save configuration data from/to a TOML file.
